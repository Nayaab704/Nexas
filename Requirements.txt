1. Initialize the project by running the following command :

    npm init 
     
    (A file called node-modules will be created inside the project folder.

2. Install following packages, below is the command to install them:

    npm install express
    npm install mongoose 
    npm install bcrypt
    npm install cors

3. Run the server:

   node server.js

4. Install Rest Client 

5. Run the following command in the CMD or any other terminal like VScode or harper

   npx create-react-app <User-defined name for the frontend>

6. Run the following command in the terminal:

   npm install react-router-dom

7. To run the React App

   npm start
